== Installation ==

You can find, download and install Comments Avatar Lazyload directly from the **Plugins** section in WordPress.

If you want to install manually, download and unzip the `comments-avatar-lazyload` file and upload to the `/wp-content/plugins/comments-avatar-lazyload/` directory. Then activate the plugin through the **Plugins** section in WordPress.
